## Install

```bash
opencode plugin roblox-opencode
```

Then run `/setup-game` in your project.

## What it does

Without this plugin, AI assistants treat Roblox like generic Lua. They miss session locking, write exploitable remotes, ignore mobile players, and produce UI that breaks on phones.

With it, the AI knows game architecture, data persistence, reactive UI, remote security, monetization, analytics, and 13 documented production footguns to avoid.

## Skills

17 on-demand skills covering language, UI (Fusion 0.3), VFX, networking, data, testing, architecture, runtime, monetization, analytics, code review, debugging, and publishing.

## Vendor libraries

Fusion 0.3, ProfileStore, Promise, Signal, Trove, Comm, Component, t, TestEZ, and 25 RbxUtil packages. Copied to your project on setup. No Wally required.

## Commands

`/setup-game` — one-time project setup
`/sync-check` — verify Script Sync
